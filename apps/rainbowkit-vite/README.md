# RainbowKit Vite Example

Vite + React example for connecting wallets on Berachain Bepolia with RainbowKit and wagmi.

## Setup

Copy `.env.example` to `.env` and set your WalletConnect project ID:

```bash
cp .env.example .env
```

```bash
VITE_WALLETCONNECT_PROJECT_ID=your_project_id
```

Install dependencies and run the app:

```bash
bun install
bun run dev
```

Use `bun run build` for a production build.
